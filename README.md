
<h1 align="center">Bolsi</h1>
<p align="center">
Aplicación de gestión de gastos personales.
</p>

 
## Descargar
 
Última versión descargable desde [GitHub](https://github.com/manucabral/Bolsi/releases).

> Compatible con Windows 10/11
> 
> No requiere instalación · Solo ejecutar
 
---
 
## Incluye
 
- **Finanzas** — registrá ingresos y gastos por categoría, con balance mensual.
- **Créditos** — seguimiento de compras en cuotas con progreso y vencimientos.
- **Notas** — diario personal con búsqueda.
- **Multi-usuario** — varios usuarios en el mismo PC, cada uno con sus datos.

---
 
## Desarrollo local
 
```bash
# Clonar
git clone https://github.com/manucabral/Bolsi.git
cd bolsi
 
# Instalar dependencias
pip install -r requirements.txt
cd frontend && npm install && npm run dev
 
# Ejecutar
python main.py
```
